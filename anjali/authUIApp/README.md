# Login Screen UI (React Native)

A small, focused React Native project that demonstrates a clean and reusable login screen UI.

**What this repo contains:** a simple login form component, input validation utilities, and styles to help you integrate a polished login screen into your app.

**Demo:** Launch the app on a simulator or device to see the login UI and basic validation in action.

**Features**

- Clean, responsive login form
- Reusable input components
- Basic validation helpers

## Quick Start

Prerequisites:

- Node.js and npm or Yarn
- React Native CLI setup (Android SDK or Xcode for iOS): follow the official guide: https://reactnative.dev/docs/environment-setup

Install dependencies:

```bash
npm install
```

Start Metro (JS bundler):

```bash
npm start
```

Run on Android:

```bash
npx react-native run-android
```

## Project Structure

- `App.tsx` — app entry point
- `src/components` — UI components (`FormInput.js`, `LoginPage.js`)
- `src/utils` — helpers and validation (`Validation.js`)
- `src/assets` — styles and static assets
- `android` / `ios` — native platform projects

## Customize

- Update `src/components/LoginPage.js` to change form fields or layout.
- Modify `src/assets/css/style.js` to tweak colors and spacing.
